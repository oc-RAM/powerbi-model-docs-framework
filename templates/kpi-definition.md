## KPIs

|---------|-----------|---------|--------|----------------------------|-----------|-------|-----------------|
| | | | | | | | |
| | | | | | | | |

> **Definition**: A plain-language explanation of what this KPI measures and why it matters.
>
> **Measure**: The measure name in the semantic model that calculates this KPI.
>
> **Target**: The numerical goal or benchmark (e.g., "> 90%", "< 24 hours", "> 4.0 / 5.0").
>
> **Threshold**: The RAG (Red/Amber/Green) boundaries. Example: Red < 80%, Amber 80-90%, Green > 90%.
>
> **Frequency**: How often this KPI is reviewed (daily, weekly, monthly, quarterly).
>
> **Owner**: The business stakeholder accountable for this KPI.
>
> **Source of Truth**: Where the official target or definition comes from (contract, SLA document, management decision, etc.).
